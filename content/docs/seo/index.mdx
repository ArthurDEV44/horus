# SEO

Cette catégorie regroupe **7 agents** spécialisés dans l'optimisation pour les moteurs de recherche.

## Agents disponibles

| Agent | Description |
|-------|-------------|
| [Metadata Expert](/docs/seo/seo-metadata-expert) | Optimisation des meta tags et Open Graph |
| [Content Expert](/docs/seo/seo-content-expert) | Structure et optimisation du contenu |
| [Schema Expert](/docs/seo/seo-schema-expert) | Données structurées JSON-LD |
| [Performance Expert](/docs/seo/seo-performance-expert) | Core Web Vitals et performance |
| [Image Expert](/docs/seo/seo-image-expert) | Optimisation des images pour le SEO |
| [Crawl Expert](/docs/seo/seo-crawl-expert) | Robots.txt, sitemap, crawlabilité |
| [Rendering Expert](/docs/seo/seo-rendering-expert) | SSR, SSG et stratégies de rendu |

## Cas d'utilisation

Ces agents sont utiles pour :

- Améliorer le classement dans les moteurs de recherche
- Optimiser les Core Web Vitals
- Implémenter les données structurées
- Configurer le crawl et l'indexation
- Choisir la stratégie de rendu adaptée
