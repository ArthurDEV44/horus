# Experts

Cette catégorie regroupe **7 agents** spécialisés dans des technologies ou domaines spécifiques.

## Agents disponibles

| Agent | Description |
|-------|-------------|
| [Architect Expert](/docs/experts/architect-expert) | Architecture globale d'applications |
| [Next.js Expert](/docs/experts/nextjs-expert) | Framework Next.js et App Router |
| [React Expert](/docs/experts/react-expert) | Bibliothèque React et écosystème |
| [Tailwind Expert](/docs/experts/tailwind-expert) | Framework CSS Tailwind |
| [Clerk Expert](/docs/experts/clerk-expert) | Authentification avec Clerk |
| [Security Expert](/docs/experts/security-expert) | Sécurité des applications web |
| [Techwatch Expert](/docs/experts/techwatch-expert) | Veille technologique |

## Cas d'utilisation

Ces agents sont utiles pour :

- Concevoir l'architecture d'une nouvelle application
- Implémenter des fonctionnalités Next.js avancées
- Optimiser les performances React
- Styliser avec Tailwind CSS
- Intégrer l'authentification Clerk
- Sécuriser les applications
- Rester à jour sur les technologies
