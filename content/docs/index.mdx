# Claude Code Agents Library

Bienvenue dans la librairie d'agents optimisés pour Claude Code.

## Catégories disponibles

Cette collection comprend **29 agents** répartis en 4 catégories :

- **Architecture** - 8 agents pour la structure et les patterns de code
- **SEO** - 7 agents pour l'optimisation du référencement
- **Tailwind Responsive** - 8 agents pour le design responsive
- **Experts** - 6 agents spécialisés (Next.js, React, Clerk, etc.)

## Comment utiliser

Chaque agent est documenté avec :

1. **SKILL** - Description des compétences et capacités
2. **REFERENCE** - Documentation technique de référence
3. **WORKFLOWS** - Flux de travail et cas d'utilisation

## Démarrage rapide

Explorez les différentes catégories dans la sidebar pour découvrir les agents disponibles.
