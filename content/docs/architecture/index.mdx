# Architecture

Cette catégorie regroupe **8 agents** spécialisés dans l'architecture logicielle et les bonnes pratiques de développement.

## Agents disponibles

| Agent | Description |
|-------|-------------|
| [Patterns Expert](/docs/architecture/patterns-expert) | Design patterns React et Backend (Container/Presentational, Factory, Repository) |
| [SOLID Expert](/docs/architecture/solid-expert) | Principes SOLID appliqués à TypeScript et React |
| [Structure Expert](/docs/architecture/structure-expert) | Organisation des fichiers et dossiers |
| [Modularity Expert](/docs/architecture/modularity-expert) | Découpage modulaire et boundaries |
| [Coupling Expert](/docs/architecture/coupling-expert) | Gestion du couplage et des dépendances |
| [State Expert](/docs/architecture/state-expert) | Gestion d'état (local, global, server state) |
| [Types Expert](/docs/architecture/types-expert) | Typage TypeScript avancé |
| [Debt Expert](/docs/architecture/debt-expert) | Identification et réduction de la dette technique |

## Cas d'utilisation

Ces agents sont utiles pour :

- Concevoir une architecture scalable
- Refactorer du code legacy
- Appliquer les design patterns appropriés
- Améliorer la maintenabilité du code
- Réduire le couplage entre modules
